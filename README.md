# Wuju CobbleBreed

##  About

**Wuju CobbleBreed** is a virtual breeding mod for Cobblemon, built using NeoForge and designed for private servers.

It provides a complete breeding system through interactive menus, including:

* Biological compatibility-based breeding
* Search system by:

  * Species
  * Name
  * IVs
  * Gender
  * Nature
* Support for tags such as:

  * F1, F2, F3 (for Pokémon with multiple maxed stats)

## Features

* Full menu-based breeding system
* Optional integration with LuckPerms
* Configurable tier system (permission-based)
* Fully customizable breeding difficulty and balance
* Admin & utility commands:

  * Check egg hatch status
  * Check Pokémon stats
  * Check player tier
  * Manual hatch command
  * Real-time config changes in-game

## Custom Held Item System

Includes an exclusive custom held item system:

* Create multiple item variations
* Configure activation chances
* Define custom effects
* Assign effects to specific items
* Durability support (items can degrade on use)

# Wuju CobbleBreed

## Sobre

O **Wuju CobbleBreed** é um mod de breed virtual para Cobblemon, desenvolvido em NeoForge e projetado para uso em servidores privados.

O mod adiciona um sistema completo de breeding através de menus interativos, incluindo:

* Seleção biológica compatível entre Pokémon
* Barra de pesquisa por:

  * Espécie
  * Nome
  * IV
  * Sexo
  * Nature
* Suporte a siglas como:

  * F1, F2, F3 (para Pokémon com múltiplos status maximizados)

## Funcionalidades

* Sistema completo de breeding via interface
* Integração opcional com LuckPerms
* Sistema de tiers configurável (integrável com permissões)
* Configuração total de balanceamento (facilitar ou dificultar gameplay)
* Comandos administrativos e utilitários:

  * Verificar status de ovos (hatch)
  * Verificar status de Pokémon
  * Verificar tier
  * Comando de hatch manual
  * Alteração de configurações em tempo real

## Sistema de Held Items Custom

O mod inclui um sistema exclusivo de held items customizados:

* Criação de múltiplas variações de itens
* Definição de chances de ativação
* Definição de efeitos aplicados
* Associação de efeitos a itens específicos
* Suporte a durabilidade (consumo do item ao usar)
